 -ApkEditor
 
 - Tutorial ENGLISH
 
 -Decompile, edit, and compile with Apk editor, free and simple.
 -Available only for Windows (7, 8, 8.1, 10)
 
 -Needed to install: Java7: 64bits: http://download.oracle.com/otn-pub/java/jdk/7u79-b15/jre-7u79-windows-x64.exe 32 Bits: http://download.oracle.com/otn-pub/java/jdk/7u79-b15/jre-7u79-windows-i586.exe
 
 -Hot to Use:
1. Run Command Prompt (CMD)
2 Installing framework. apktool if APP-NAME.apk
3 Decomping. java -jar apktool.jar d -s APP-NAME.apk
4 Compiling. java -jar apktool.jar b APP-FOLDER-NAME
5 Signing. java -jar signapk.jar certificate.pem key.pk8 APP-NAME.apk NEW-APP-NAME.apk
 
 - Tutorial Português
 
 - Descompile, edite e compile com o Apk Editor, é Grátis e Simples.
 - Disponivel somente para Windows (7, 8, 8.1, 10)
 
 Necessario instalar: Java7: 64bits: http://download.oracle.com/otn-pub/java/jdk/7u79-b15/jre-7u79-windows-x64.exe 32 Bits: http://download.oracle.com/otn-pub/java/jdk/7u79-b15/jre-7u79-windows-i586.exe
 
 -Como usar:
 
 1. Execute o Prompt de Comando (CMD)
 2 Instalando framework. apktool if NOME-DO-APP.apk
 3 Descompilando. java -jar apktool.jar d -s NOME-DO-APP.apk
 4 Compilando. java -jar apktool.jar b NOME-DA-PASTA-DO-APP
 5 Assinando. java -jar signapk.jar certificate.pem key.pk8 NOME-DO-APP.apk NOME-DO-NOVO-APP.apk
